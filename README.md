# fsmp
FSMP - Fucking Simple Music Player

A simple music/audio player with the following features:

- Play/Stop function (the purpose is to listen music tracks from start to finish)
- Next/Previous music in playlist
- Add files to a playlist
- Shuffle playlist
- Sound enhancements

A great tool if you want to load a big playlist and play all day long without complications (ambient music for public spaces) and have some special sound enhancements.

Requires:

- python3
- ffmpeg + ffplay
- gtk libraries

How to use:

- Download the fsmp.py or copy the code to a text file and save it with .py extension.
- Make it executable with "chmod +x player.py"
- Run ./fsmp.py in terminal or open it in you desktop environmet.

Hope you like it!
